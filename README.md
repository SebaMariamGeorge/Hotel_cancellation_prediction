# 🏨 Hotel Cancellation Prediction System

A machine learning-based hotel reservation system developed using **Python, Flask, MySQL, HTML, CSS, JavaScript, and Scikit-learn**.

## 🚀 Features

### User Module
- User Registration & Login
- Room Reservation
- Automatic Room Allocation
- View Multiple Reservations
- Booking Confirmation

### Staff Module
- Staff Login
- Dashboard
- Cancellation Prediction
- Overbooking Prediction

## 🛠 Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- Pandas
- NumPy
- Scikit-learn

## 📂 Project Structure

```
Hotel_cancellation_prediction
│
├── app.py
├── templates/
├── static/
├── Hotel_Cancellation_ML.ipynb
└── requirements.txt
```

## ▶️ How to Run

1. Install requirements

```
pip install -r requirements.txt
```

2. Configure MySQL

3. Run

```
python app.py
```
